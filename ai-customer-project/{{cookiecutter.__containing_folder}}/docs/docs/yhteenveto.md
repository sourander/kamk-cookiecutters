# Yhteenveto

Projektin yhteenvedossa kuvataan lyhyesti projektin eteneminen, esitellään keskeisimmät tulokset ja pohditaan jatkokehityskohteita. Kiireisen lukijan tulisi ymmärtää työsi avainkohdat lukemalla Johdanto ja Yhteenveto -osiot. Mikäli hän kiinnostuu, hän voi syventyä tarkemmin data-aineistoon, menetelmiin ja tuloksiin projektin muissa osioissa.

## Tuntikirjanpito
